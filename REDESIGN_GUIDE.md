## 🐛 Troubleshooting

**Issue**: Changes not showing after push?
- **Solution**: Hard refresh your browser (Ctrl + Shift + R)
- **Wait**: GitHub Pages can take up to 10 minutes
- **Check**: Visit https://github.com/briprogramming/Portfolio/actions to see build status

**Issue**: Colors look different than expected?
- **Solution**: Some browsers cache CSS aggressively. Clear cache or use incognito mode

**Issue**: Want to go back to original?
- **Solution**: Run `git log` to see commits, then `git revert <commit-hash>`
